Common information for all platforms with technical description:

  • What it is ?
    Pulse Secure is the newest 4Finance VPN solution for un-managed devices connection to 4Finance resources.
  • What it replaces ?
    It is substitute  to previously used Sophos and TMG VPNs.
  • What are the benefits ?
    • One platform for managed and un-managed devices.
      • Split access zones based on managed/un-managed computer. 
      • WHAT RESOURCES ARE AVAILABLE FROM UN-MANAGED DEVICES terminal servers like - dcm1-wts-00(4,6,7), linux-gw.4finance.net and basic services required for daily work (like wiki, jira, zabbix etc).
    • Better security (MFA) - multi factor authentication to ensure protection against compromised devices or user credentials.
    • Redundancy - VPN gateways are spread across the world ensuring redundancy and reduced distance to 4finance resources depending on your location
    • Split tunnel  - better network speed to access non-4finance resources
    • Easy to connect with mobile devices
    • Multi platform vendor support (Win, Linux, MAC, IOS, Android)

Step #1 - Set up Multi Factor Authenticator (MFA)

Why this is required ?

If you are using a 4Finance corporate computer, you do not require this step, please find Step #2 "Corporate 4Finance Windows computer".

Multi Factor Authentication is required for security reason, in case when your device is compromised nobody is able to connect to 4Finance resources without MFA token.

What are possible authenticators ?

You can use Google Authenticator or Microsoft Authenticator application. Both applications are available in IOS and Android Marketplace and you can download / install it on your smartphone or tablet.

How to activate MFA on your smartphone or tablet ?

  1. Install Authenticator application on your smartphone or tablet.

  2. Depending on your region (Europa, Latin America), open these link in your web browser:

    Europe region - https://pulse.4finance.com/nm
    Latin America region - https://us-pulse.4finance.com/nm

  3. Log in with your Active Directory credentials and read the instructions on the page.
  4. Open the Authenticator app and add a "Work or School account", after which you will be prompted to scan the QR code, which you see on the page.
  5. After MFA is activated close web link and proceed with client configuration (Step #2)

Step #2 - Set up Client

Un-managed/Private Windows computer

 Click here to expand...

1. Make sure you have registered your MFA account (Step #1)

2. Download from links below "Pulse Secure" client package:

Windows 32-Bit or Windows 64-bit

3. Install package on your workstation

4. Set up connection parameters:

Using Pulse Secure Client user interface add new connection by clicking on Add (+) option on the top-right-hand corner of the main Pulse Secure client screen

Enter necessary connection details:

In the Server URL field type:
For Europe region connection: https://pulse.4finance.com/nm
For Latin America region connection: https://us-pulse.4finance.com/nm

6. Press Connect, enter your corporate credentials:

7. Enter the token code from the MFA application on the phone:

8. Pulse secure is now connected:



Corporate 4Finance Windows computer

 Click here to expand...

1. There is no need to register MFA, if using a corporate computer provided by 4Finance (skip Step #1)

2. Find Pulse Secure in Start menu and launch the application:


3. Now simply press "Connect" and it will automatically establish VPN connection. No additional steps required:


Linux (Native way)

 Click here to expand...

1. Make sure you have registered your MFA account (Step #1)


2. Using package manager of your distro, make sure, you have the following packages installed:

- NetworkManager-openconnect >= 1.2.4

- openconnect (see patched packages below)

openconnect package in current released version, has a bug, leading to wrong interpretation of TOTP form, so most probably, you'll need to installed patched (patch is in comments) version of the package.

Fedora 27+

openconnect-7.08-5.4f.fc27.x86_64.rpm

Ubuntu >= 18.04 / Debian / Mint >= 19 (if you have Mint 18, the only way to use native setup is upgrade to 19)

openconnect_7.08-3.1_amd64.deblibopenconnect5_7.08-3.1_amd64.deb


3. Open network settings in Gnome and add new VPN connection (should be similar for all NetworkManager GUIs)

4. Choose "Cisco AnyConnect Compatible VPN (openconnect)

5. Set up connection parameters as shown on the screenshot below:


  1. In the Name field type - 4Finance VPN
  2. In the VPN Protocol field choose - Juniper/Pulse Network Connect
  3. In the Gateway type:
    For Europe region connection: pulse.4finance.com/nm
    For Latin America region connection: us-pulse.4finance.com/nm
  4. In the Token Mode filed choose - TOTP manually entered

5. Connect to the newly created VPN connection. It will ask for your password and MFA code and connect you to VPN



Linux (Pulse Secure client)

 Click here to expand...

1. Make sure you have registered your MFA account (Step #1)


2. Download from links below "Pulse Secure" client package:

Debian based installation (Ubuntu, Debian, ..) - 32-bit, 64-bit

RPM based installation (CentOS, RHEL, ..) - 32-bit, 64-bit


3. Install package on your workstation.

Example: (Ubuntu-16.04.4 LTS and 64bit installation package)

~$ sudo dpkg -i pulse-5.3R3.x86_64.deb

~$ /usr/local/pulse/PulseClient_x86_64.sh install_dependency_packages

  • Workaround for missing packages in Fedora 27/28
  • On Ubuntu 18 (at least on xfce desktop) pulse client fails silently due to missing dependencies. No errors are shown. To workaround please execute: sudo apt-get install libwebkitgtk-1.0


4. Set up connection parameters:

Using Pulse Secure Linux Client user interface add new connection by clicking on Add (+) option on the top-right-hand corner of the main Pulse Secure Linux client screen,

where:

  1. In the Name field type - 4Finance VPN
  2. In the URL field type:
    For Europe region connection: https://pulse.4finance.com/nm
    For Latin America region connection: https://us-pulse.4finance.com/nm

5. On the newly created profile click "Connect". It will ask for your password and MFA code and connect you to VPN:





Mac

 Click here to expand...

1. Make sure you have registered your MFA account (Step #1)

2. Download from link below "Pulse Secure" client package:

Download MAC .dmg package


3. Install package on your workstation

4. Open Pulse Secure from the Application List and Select "Open Pulse Secure" from the tray:


5. Set up connection parameters:

Using Pulse Secure Client user interface add new connection by clicking on Add (+) option on the bottom-left-hand corner of the main Pulse Secure client screen

In the Server URL field type:

For Europe region connection: https://pulse.4finance.com/nm
For Latin America region connection: https://us-pulse.4finance.com/nm


6. Press "Connect" and enter corporate credentials, also enter the token code from the MFA application on the phone:



7. Pulse Secure is now connected:

2. Download from App Store "Pulse Secure" application

In the URL field type:

For Europe region connection: https://pulse.4finance.com/nm
For Latin America region connection: https://us-pulse.4finance.com/nm


5. On the newly created profile click "Connect". It will ask for your password and MFA code and connect you to VPN:

    Hints to build .deb packages:

    1. mkdir openconnect && cd openconnect
    2. "enable source repos"
    3. sudo apt install dpkg-dev
    4. apt source openconnect
    5. sudo apt build-dep openconnect
    6. cd openconnect-7.08
    7. sudo apt install devscripts quilt
    8. dch --nmu
    9. cd debian/patches
    10. quilt import -P fix-openconnect-mfa.patch /tmp/fix-openconnect-mfa.patch
    11. quilt refresh
    12. cd ../../
    13. debuild -us -uc
    14. PROFIT
